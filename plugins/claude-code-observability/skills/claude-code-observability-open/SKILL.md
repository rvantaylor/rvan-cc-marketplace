Open the local Arize Phoenix UI.

Run: `open http://localhost:6006`

Note: `open` is macOS-only. On Linux, use `xdg-open http://localhost:6006` instead.

If the command fails or the browser shows a connection error, suggest running the claude-code-observability-status skill to check whether Phoenix is running.
